Подключение к серверу


Подключаемся к учебному серверу kubsu-dev.ru по SSH с помощью клиента Putty


IP-адрес kubsu.ru - команда ping


С помощью команды ping на учебном сервере узнаем IP-адрес веб-сервера kubsu.ru. Проверяем соединение с сервером.


А-запись и МХ-запись kubsu.ru - nslookup


С помощью команды nslookup узнаем A-записи и MX-записи домена kubsu.ru. С помощью А-записи домен прикрепляется к IP-адресу, т.е. А-запись позволяет найти IP. При создании электронной почты в ресурсных записях прописывают MX-записи.


А-запись и МХ-запись kubsu-dev.ru - nslookup


С помощью команды nslookup узнаем A-записи и MX-записи домена kubsu-dev.ru. С помощью А-записи домен прикрепляется к IP-адресу, т.е. А-запись позволяет найти IP. При создании электронной почты в ресурсных записях прописывают MX-записи.


Даты регистрации доменов - whois


С помощью команды whois узнаем дату регистрации домена kubsu.ru и kubsu-dev.ru
